0.10.0b [2020-01-17]
+ Доработан дизайн меню.
+ Добавлен кулдаун в 1 секунду для отправки пакетов - операций в меню почты.
+ Обновлены параметры конфигурации:
 Клиент 
 - "enable_mail_menu_key" [boolean] - разрешить добавление клавиши "открыть меню почты"
 
 Сервер 
 - "mailbox_size" [int](синхронизируется) - размер почтового ящика
 - "system_letter_expire_time_hours" [int](синхронизируется) - срок хранения письма отправленного системой в часах
 - "letter_expire_time_hours" [int](синхронизируется) - срок хранения письма отправленного игроком в часах
 - "system_remittance_expire_time_hours" [int](синхронизируется) - срок хранения денежного перевода отправленного системой в часах
 - "remittance_expire_time_hours" [int](синхронизируется) - срок хранения денежного перевода отправленного игроком в часах
 - "system_package_expire_time_hours" [int](синхронизируется) - срок хранения посылки отправленной системой в часах
 - "package_expire_time_hours" [int](синхронизируется) - срок хранения посылки отправленной игроком в часах
 - "package_with_cod_expire_time_hours" [int](синхронизируется) - срок хранения посылки с наложенным платежом в часах
 - "mail_sending_cooldown_seconds" [int] - кулдаун отправки почты
 - "remittance_max_value" [long](синхронизируется) - максимальная сумма денежного перевода
 - "package_max_amount" [int](синхронизируется) - максимальное количество предметов в посылке
 - "package_with_cod_max_value" [long](синхронизируется) - максимальная сумма наложенного платежа
 - "letter_postage_value" [long](синхронизируется) - денежный сбор за отправку письма
 - "remittance_postage_percent" [int](синхронизируется) - денежный сбор за отправку денежного перевода (процент от суммы перевода)
 - "package_postage_value" [long](синхронизируется) - денежный сбор за отправку посылки
 - "package_with_cod_postage_percent" [int](синхронизируется) - комиссия за продажу предмета наложенным платежом (процент от стоимости)
 
+ Добавлены команды:
 Сервер - Операторы [/oxygenop]
 - Отправить письмо: /oxygens mail -send <target> <mail type> <subject> <message> <args...>
 
 Клиент [/oxygenc]
 - Открыть меню почты: /oxygenc mail
 - Сбросить данные клиента: /oxygenc mail -reset-data

+ Добавлены привилегии:
 - "mail:mailboxSize" [id - 800, type - INT] - определяет размер почтового ящика
 - "mail:allowMailSending" [id - 801, type - INT] - возможность отправлять почту. По умолчанию любой игрок может отправлять почту.
 - "mail:mailSendingCooldownSeconds" [id - 802, type - INT] - кулдаун отправки писем
 
 - "mail:remittanceMaxValue" [id - 820, type - LONG] - максимальная сумма денежного перевода
 - "mail:packageMaxAmount" [id - 821, type - INT] - максимальное количество предметов в посылке
 - "mail:packageWithCODMaxValue" [id - 822, type - LONG] - максимальная сумма для посылки наложенным платежом
 
 - "mail:letterPostageValue" [id - 830, type - INT] - денежный сбор за отправку письма
 - "mail:remittancePostagePercent" [id - 831, type - INT] - денежный сбор за отправку денежного перевода (процент от суммы перевода)
 - "mail:packagePostageValue" [id - 832, type - LONG] - денежный сбор за отправку посылки
 - "mail:packageWithCODPostagePercent" [id - 833, type - INT] - комиссия за продажу предмета наложенным платежом (процент от стоимости)
 
+ Добавлены клиентские настройки:
 Общие
 - возможность открыть меню почты через общее меню (Oxygen Menu)
 
 Интерфейс
 - расположение меню
